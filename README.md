# KonfioCodeChallenge!

Android Code Challenge que implementa la API  [https://jsonblob.com/api/1151549092634943488](https://jsonblob.com/api/1151549092634943488) , utilizando la arquitectura MVI.

# Stack
- Kotlin
- Androidx Lifecycle
- kotlin Flows
- Retrofit2
- Compose
- Coil
- Dagger Hilt
- Room Database
